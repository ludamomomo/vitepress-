# WRYTooltip

how to use

```md
<template>
    <Tooltip text="我是悬停的信息" position="right">
      <button>悬停提示信息的</button>
    </Tooltip>
</template>

## 属性
text="内容区域"
position="位置"(right left top button)

## 功能
当你的鼠标在button区域内的时候 它会弹出一个消息区域 
这个消息是你可以自定义的 同时消息展示的位置也是可以灵活的变化
这里提供了上下左右四个方位的位置属性
